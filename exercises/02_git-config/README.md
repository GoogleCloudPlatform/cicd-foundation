### Customize

<details>
<summary>git config</summary>

You may find some of the following configs useful:
```
[alias]
        a = add
        br = branch
        c = commit
        ca = commit --amend
        cm = commit -m
        co = checkout
        d = diff
        gl = config --global -l
        l = log
        last = log -1 HEAD
        p = push
        s = status -sb
        se = !git rev-list --all | xargs git grep -F
        st = status -sb
        sw = switch
        unstage = reset HEAD --
[init]
        defaultBranch = main
```
</details>
